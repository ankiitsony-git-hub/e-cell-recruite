# SkillBridge : Startup Landing Page

## Project Overview

SkillBridge is a simple startup landing page that shows an idea for a platform where college students can find freelance work and startups can hire students with useful skills.

Many students learn things like coding, design, writing, or digital marketing in college. But most of the time they do not get a chance to use these skills in real projects. At the same time, many small startups need help with small tasks but cannot always hire expensive professionals.

SkillBridge tries to solve this problem by connecting students and startups on one platform. Students can create a profile, show their skills, and apply for projects posted by startups.

This landing page explains the idea of the startup through different sections like Home, About Us, Problem Statement, Product Solution, and Contact.



## Technologies Used

This project was built using basic web development tools:

HTML : used to create the structure of the website  
CSS : used to style the page and make it look better  
JavaScript : used to add simple interactions and animations  

No frameworks or external libraries were used.



## Features

--> Home section with startup name and short introduction  
--> About section that explains the idea of the startup  
--> Problem section describing the issue students face  
--> Solution section explaining how SkillBridge works  
--> Animated statistics on the page  
--> FAQ section where users can click to see answers  
--> Contact form where users can send a message  
--> Smooth scrolling and small animations



## Instructions to Run the Project

### Run Using GitHub

1. Go to the GitHub repository.
2. Click the **Code** button.
3. Download the project as a **ZIP file**.
4. Extract the ZIP file on your computer.
5. Open the extracted folder.
6. Double click the **index.html** file to open it in your web browser.

### Run Using VS Code

1. Clone the repository using the terminal:

2. Open **VS Code**.

3. Click **File → Open Folder**.

4. Select the project folder.

5. Open the file **index.html** and run using code runner.
